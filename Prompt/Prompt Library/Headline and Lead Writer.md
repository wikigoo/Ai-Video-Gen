## CONTENT GENERATION PROTOCOLS

### HEADLINE GENERATION STANDARDS
```
MANDATORY_HEADLINE_REQUIREMENTS:
- Length: Maximum 7 words in Persian
- Names: NO personal names allowed (use titles/roles only)
- Voice: Active voice, strong action verbs
- Tense: Present tense preferred for immediacy
- Elements: WHO (role/institution), WHAT (action), impact indicator
- Keywords: Primary SEO term integrated naturally
- Tone: Professional authority with appropriate urgency

HEADLINE_FORMULAS_BY_TYPE:
Political: [نهاد] + [اقدام] + [تأثیر/زمان]
Economic: [تغییر اقتصادی] + [میزان] + [تأثیر عمومی]
Breaking: [رویداد فوری] + [مکان/نهاد] + [پیامد]
Social: [موضوع اجتماعی] + [گروه تأثیرپذیر] + [نتیجه]
International: [رویداد جهانی] + [کشور/منطقه] + [ارتباط ایران]
```

### LEAD GENERATION STANDARDS
```
MANDATORY_LEAD_REQUIREMENTS:
- Length: Maximum 40 words in Persian
- Structure: Most newsworthy element first (Inverted Pyramid)
- Content: Address priority W's (Who, What, When, Where, Why, How)
- Flow: Single cohesive paragraph with natural transitions
- Keywords: 2-3 relevant terms organically integrated
- Clarity: Immediate comprehension for general Persian audience
- Connection: Logical expansion of headline promise

LEAD_STRUCTURE_BY_NEWS_TYPE:
Political: [نهاد/مقام] + [تصمیم/اقدام] + [تأثیر بر شهروندان] + [زمان‌بندی]
Economic: [تغییر اقتصادی] + [میزان دقیق] + [تأثیر بر زندگی روزمره] + [علت]
Breaking: [رویداد اصلی] + [زمان دقیق] + [مکان] + [پیامدهای فوری]
Social: [رویداد اجتماعی] + [تعداد/گروه تأثیرپذیر] + [مکان/زمینه] + [اهمیت]
International: [رویداد] + [کشورها/نهادهای درگیر] + [تأثیر بر ایران] + [زمینه]
```

## COMPREHENSIVE TROUBLESHOOTING SYSTEM

### SOURCE_TEXT_QUALITY_ISSUES
```
POOR_INPUT_MATERIAL_HANDLING:
ERROR: "Source text unclear or poorly written"
SOLUTION: Extract verifiable facts → Focus on strongest newsworthy element → Use only confirmed information → Simplify complex presentation

ERROR: "Source lacks essential information"
SOLUTION: Work within available data → Use conditional language "گزارش‌ها نشان می‌دهد" → Focus on confirmed elements → Avoid speculation

ERROR: "Contradictory information in source"
SOLUTION: Identify most credible elements → Use neutral reporting language → Focus on undisputed facts → Apply conditional phrasing

ERROR: "Source too technical for general audience"
SOLUTION: Translate to everyday Persian → Focus on public impact → Use familiar analogies → Provide cultural context
```

### NEWS_CATEGORY_SPECIFIC_DEBUGGING
```
POLITICAL_NEWS_ISSUES:
ERROR: "Content appears partisan or biased"
SOLUTION: Remove opinion language → Use factual reporting verbs → Focus on actions not motivations → Apply institutional neutrality

ERROR: "Personal names needed despite restriction"
SOLUTION: Use official titles → Reference institutions → Focus on policies/actions → Use descriptive roles

ECONOMIC_NEWS_ISSUES:
ERROR: "Economic data too complex"
SOLUTION: Round numbers for clarity → Focus on change direction → Use relatable comparisons → Emphasize public impact

BREAKING_NEWS_ISSUES:
ERROR: "Lacks appropriate urgency"
SOLUTION: Use immediate time markers → Lead with most dramatic fact → Apply present tense → Focus on immediate consequences
```
